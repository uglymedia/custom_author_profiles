Custom Author Profiles is a simple plugin that allows you to add additional fields to user/author profiles.

== Description ==

Custom Author Profiles simplifies the process of adding custom information to a user or author account. The plugin allows you to quickly add custom fields to a user edit page without complicated configuration. The new fields are automatically included in the user edit page once they are created and are easily inserted into Wordpress template files.

== Installation ==

Upload the plugin to Wordpress, enable it, and start adding new fields!
